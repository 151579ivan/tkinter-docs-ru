44. ttk.Sizegrip

Используйте этот виджет для того, чтобы предоставить возможность пользователю изменять размер всего окна приложения. Обычно этот виджет располагается в правом нижнем углу приложения. Обязательно сделайте всё приложение изменяемым по размеру, используя приемы, описанные в Раздел 4.3, "Настройка размеров столбцов и строк" и Раздел 4.4, "Изменение размеров корневого окна".

Чтобы создать ttk.Sizegrip в качестве дочернего элемента данного виджета parent:

w = ttk.Sizegrip(parent, option=value, ...)

Таблица 62. Опции ttk.Sizegrip
class_ Имя класса виджета. Оно может быть указано при создании виджета, но не может быть изменено позже. Для объяснения классов виджетов смотрите Раздел 27, "Стандартизация внешнего вида".
style Стиль, который будет использоваться при отрисовке этого виджета; см. раздел Раздел 49, "Использование и настройка стилей ttk". Единственная характеристика стиля, которую вы можете настроить, это background, которая задает цвет виджета.
