24.3. Изображения виджета Текст

В текстовый виджет можно поместить изображение или битмап. Оно рассматривается как один символ, размер которого равен естественному размеру объекта. См. Раздел 5.9, "Изображения" и Раздел 5.7, "Битовые изображения".

Изображения помещаются в текстовый виджет путем вызова метода .image_create() этого виджета. Последовательность вызова и другие методы для работы с изображениями см. ниже.

Изображениями можно манипулировать, передавая их имена методам текстового виджета. Вы можете задать Tkinter имя для изображения или просто позволить Tkinter сгенерировать имя по умолчанию для этого изображения.

Изображение может появляться любое количество раз в одном и том же виджете Text. Каждый экземпляр будет иметь уникальное имя. Это имя можно использовать в качестве индекса.
