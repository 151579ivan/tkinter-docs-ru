5.11. Имена окон

Термин окно описывает прямоугольную область на рабочем столе.

  • Верхнее (top-level) или корневое (root) окно - это окно, которое имеет независимое существование под менеджером окон. Оно оформлено декорациями менеджера окон и может быть перемещено и изменено в размерах независимо. Ваше приложение может использовать любое количество верхних окон.

  • Термин "окно" также применяется к любому виджету, который является частью верхнего окна.

Tkinter называет все эти окна с использованием иерархического имени пути окна.

  • Имя корневого окна - '.'.

  • Дочерние окна имеют имена в форме '.n', где n - некоторое целое число в строковой форме. Например, окно с именем '.135932060' является дочерним окном корневого окна ('.').

  • Дочерние окна внутри дочерних окон имеют имена в форме 'p.n', где p - имя родительского окна, а n - некоторое целое число. Например, окно с именем '.135932060.137304468' имеет родительское окно '.135932060', поэтому оно является внучатым окном корневого окна.

  • Относительное имя окна - это часть после последней '.' в имени пути. Продолжая предыдущий пример, у внучатого окна относительное имя '137304468'.

Чтобы получить путь для виджета w, используйте str(w).

Смотрите также Раздел 26, "Универсальные методы виджетов" для методов, которые можно использовать для работы с именами окон, особенно методы .winfo_name, .winfo_parent и .winfo_pathname.
