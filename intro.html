1. Кроссплатформенный конструктор графического пользовательского интерфейса для Python.

Tkinter — это набор виджетов графического пользовательского интерфейса (GUI) для Python.

Этот документ был написан для Python 2.7 и Tkinter 8.5, работающих в системе X Window под Linux. Ваша версия может отличаться.

Начнем с рассмотрения видимой части Tkinter: создания виджетов и их расположения на экране. Позже мы поговорим о том, как связать лицо — «лицевую панель» приложения с лежащей в его основе логикой.
