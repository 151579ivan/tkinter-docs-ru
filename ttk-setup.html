28.1. Импортирование ttk

Существуют различные способы импорта модуля ttk.

  • Если вы предпочитаете, чтобы все виджеты и другие возможности Tkinter и ttk находились в вашем глобальном пространстве имен, используйте эту форму импорта:

    from tkinter import *
from tkinter.ttk import *

    Важно выполнить эти два импорта именно в таком порядке, чтобы все типы виджетов из ttk заменили эквивалентные виджеты из Tkinter. Например, все ваши виджеты Button будут взяты из ttk, а не из Tkinter.

  • Вы можете использовать использовать эту форму импорта:

    import tkinter.ttk as ttk

    Таким образом, после этого импорта ttk.Label - это конструктор виджета Label, ttk.Button - это Button и так далее.

    Если вам нужно обратиться к элементам из модуля Tkinter, они доступны в виде ttk.tkinter. Например, для создания корневого окна используйте root = ttk.tkinter.Tk(), а код привязки для "northeast" (северо-восток) - это ttk.tkinter.NE.

  • Также вы можете импортировать Tkinter и ttk отдельно следующим образом:

    import tkinter as tk
from tkinter import ttk

    После такой формы импорта вам будут доступны как виджеты Tkinter (например tk.Button), так и виджеты ttk (ttk.Button).
