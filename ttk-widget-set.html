28.2. Набор виджетов ttk

Модуль ttk содержит различные версии большинства стандартных виджетов Tkinter и несколько новых. Эти виджеты заменяют одноименные виджеты из Tkinter:

  • Раздел 29, "ttk.Button".

  • Раздел 30, "ttk.Checkbutton".

  • Раздел 32, "ttk.Entry".

  • Раздел 33, "ttk.Frame".

  • Раздел 34, "ttk.Label".

  • Раздел 35, "ttk.LabelFrame".

  • Раздел 36, "ttk.Menubutton".

  • Раздел 38, "ttk.PanedWindow".

  • Раздел 40, "ttk.Radiobutton".

  • Раздел 41, "ttk.Scale".

  • Раздел 42, "ttk.Scrollbar".

Эти виджеты являются новыми и специфичными для ttk:

  • Раздел 31, "ttk.Combobox".

  • Раздел 37, "ttk.Notebook".

  • Раздел 39, "ttk.Progressbar".

  • Раздел 43, "ttk.Separator".
