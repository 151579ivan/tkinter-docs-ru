43. ttk.Separator

Используйте этот виджет для размещения горизонтальной или вертикальной линии, отделяющей другие виджеты. Виджет отображается в виде тонкой полосы шириной 2 пикселя. Обязательно используйте опцию sticky в методе .grid(), чтобы растянуть виджет, иначе он будет отображаться только как один пиксель.

Чтобы создать ttk.Separator в качестве дочернего элемента данного виджета parent:

w = ttk.Separator(parent, option=value, ...)

Опции:

Таблица 61. Опции ttk.Separator
class_ Имя класса виджета. Оно может быть указано при создании виджета, но не может быть изменено позже. Для объяснения классов виджетов смотрите Раздел 27, "Стандартизация внешнего вида".
orient Установите orient=tk.HORIZONTAL для горизонтального разделителя, orient=tk.VERTICAL для вертикального (ориентация по умолчанию).
style Стиль, который будет использоваться при отображении этой линии; см. Раздел 49, "Использование и настройка стилей ttk". Единственный параметр стиля, который можно настроить, это background, который задает цвет разделительной линии; по умолчанию используется темно-серый цвет.

Единственными методами, доступными для виджета ttk.Separator, являются методы, перечисленные в Раздел 46, "Методы, общие для всех виджетов ttk".
