8.8. Объекты битовых изображений на холсте

Объект битмапа на холсте отображается двумя цветами: цветом фона (для значений 0) и цветом переднего плана (для значений 1).

Чтобы создать объект битмапа на холсте C, используйте:

id = C.create_bitmap(x, y, option, ...)

Этот метод возвращает целочисленный идентификатор объекта изображения для данного холста.

Значения x и y представляют собой опорную точку, которая определяет местоположение битмапа.

Опции включают в себя:

Таблица 8. Опции объекта битового изображеня на холсте
activebackground
activebitmap
activeforeground		 Эти параметры определяют значения background, bitmap и foreground, когда указатель мыши находится над битовым изображением.
anchor Битовое изображение позиционируется относительно точки (x, y). По умолчанию устанавливается anchor=tk.CENTER, что означает, что битовое изображение центрируется по позиции (x, y). См. Раздел 5.5, "Якоря" для различных значений параметра anchor. Например, если указано anchor=tk.NE, битовое изображение будет позиционировано так, чтобы точка (x, y) находилась в северо-восточном (верхнем правом) углу битового изображения.
background Цвет, который будет отображаться там, где значения в битовом изображении равны 0. По умолчанию установлено background='', что означает прозрачность.
bitmap Используемое битовое изображение; см. Раздел 5.7, "Битовые изображения".
disabledbackground
disabledbitmap
disabledforeground		 Эти параметры определяют фон, битовое изображение и передний план, которые будут использоваться, когда состояние битового изображения state равно tk.DISABLED.
foreground Цвет, который будет отображаться там, где значения в битовом изображении равны 1. По умолчанию установлено foreground='black'.
state По умолчанию элементы создаются с state=tk.NORMAL. Используйте tk.DISABLED, чтобы сделать элемент серым и нереагирующим на события; используйте tk.HIDDEN, чтобы сделать элемент невидимым.
tags Если это строка, битовое изображение помечается этой строкой. Используйте кортеж строк, чтобы пометить битовое изображение несколькими тегами. См. Раздел 8.4, "Теги холста".
